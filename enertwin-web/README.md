# enertwin-web

企业管理后台，推荐 Vue 3 + TypeScript + Vite + Pinia + Element Plus。

建议模块：

- `src/views/auth`：登录认证。
- `src/views/park`：园区、楼宇管理。
- `src/views/device`：设备台账、点表、电表。
- `src/views/pv`：光伏资产管理。
- `src/views/storage`：储能资产管理。
- `src/views/charging`：充电桩管理。
- `src/views/alarm`：告警中心。
- `src/views/report`：日报、月报、低碳收益。
- `src/views/dispatch`：调度策略配置。
- `src/views/system`：用户、角色、权限。
